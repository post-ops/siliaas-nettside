## Weekly KPI Routine (30 min)

Every week:

1. Pull analytics data (last 7-14 days)
2. Fill out: `docs/weekly-kpi-review-template.md`
3. Identify scenario (A/B/C/D)
4. Choose ONE change only
5. Create PR with KPI checklist
6. Deploy
7. Review next week

⚠️ Rules:
- Do not change multiple things at once
- Do not act on low data (<100 sessions)
- Always tie changes to a KPI
